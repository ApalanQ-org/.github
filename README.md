## Getting Started

To leverage the Copilot coding agent in your projects:

1. Ensure clear issue descriptions with context
2. Provide relevant error messages or logs
3. Include expected vs actual behavior
4. Reference related code sections when applicable
5. Specify any constraints or requirements

## Best Practices

- **Be Specific**: Clear, detailed descriptions lead to better results
- **Provide Context**: Share relevant background information
- **Iterative Approach**: Complex tasks may be broken into smaller steps
- **Review Changes**: Always review and test proposed changes
- **Maintain Standards**: Ensure changes align with project guidelines
